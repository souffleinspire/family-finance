# 家庭账本 - 家庭财务 Web App

简单、美观、实用的家庭财务记账应用。

## ✨ 功能特点

- 📱 **多端适配**：响应式设计，支持手机和电脑
- 👨‍👩‍👧‍👦 **多人协作**：家庭成员各自记账，定期同步
- 📊 **数据分析**：月度收支统计、预算执行率
- 🎯 **目标管理**：买房首付、旅行基金等财务目标
- 💾 **本地存储**：IndexedDB，数据离线可用
- 🔄 **离线同步**：通过 JSON 文件导入导出实现数据同步

## 🎨 设计风格

- 极简主义，大量留白
- 低饱和度 + 高明度配色
- 32px 大圆角卡片
- 柔和渐变和阴影
- 参考 Apple + Airbnb + Notion

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000

### 构建生产版本

```bash
npm run build
```

## 📁 项目结构

```
family-finance/
├── src/
│   ├── assets/styles/     # 全局样式

├── package.json
├── tailwind.config.js

```

## 💾 数据存储

- 使用 IndexedDB 实现本地存储
- 支持数据导出/导入（JSON 格式）
- 支持完整备份/恢复

## 🔒 隐私说明

- 所有数据存储在本地设备
- 不上传到任何服务器
- 请定期导出备份数据
